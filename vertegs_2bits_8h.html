Detailed Description

Bits and bitfields.

Vertex.

This header file provides macros for operating with bits and bitfields. Use the VxBits data type for bitwise operations as it avoids many issues related to integral promotion, which is a common source of bugs when doing bit manipulations.

Note
Compiling with -Wconversion is recommended.

Usage.
