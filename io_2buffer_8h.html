struct StIo * st_iobuffer_init ( struct StIoBuffer *  io_buff)

Initialize a memory buffer stream.

In order to use the memory buffer stream interface: initialize the interface and assign the front of the buffer to its cp member.

struct StIoBuffer io_buff = {.cp = buff};
struct StIo* io = st_iobuffer_init(&io_buff);
